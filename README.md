# 筋トレマッチングアプリ

## 概要
筋トレでの合トレ仲間を見つけるためのマッチング＆チャットアプリです。

## 主な機能
- プロフィール登録・編集
- ユーザー検索、いいね、マッチング
- チャット機能
- 通知・おすすめユーザー表示

## 使用技術
- Django（Python）
- HTML（Djangoテンプレート）
- CSS/JS（Vanilla）
- MySQL（開発用DB）
- PostgresSQL（Web公開時）
